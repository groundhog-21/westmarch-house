# 4. Reenact the Script, Step by Step

1. Copy the next scripted prompt  
2. Paste it into the Streamlit chat input  
3. Press **Enter**  
4. Wait for the response  
5. Continue to the next line  
6. Repeat until the script ends

To reproduce a demo correctly, follow the script **line-for-line**.
